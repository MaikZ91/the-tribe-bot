NEUE BILDER REINSCHIEBEN
------------------------
- Leg neue Fotos einfach hier ab (JPG, PNG oder WebP ist egal).
- Sag mir danach kurz Bescheid ("neue Bilder im hero-Ordner"), dann:
    * optimiere ich sie zu WebP (klein, performant)
    * benenne sie passend (1..N.webp)
    * trage sie in docs/v2.html ein (so viele Frames wie Bilder)
- Wenn du eine bestimmte Reihenfolge willst, nummeriere die Dateien
  selbst durch (1, 2, 3, ...).

Tipp: Querformat-Fotos sehen am besten aus. Wichtiges Motiv eher
mittig, weil die Ränder je nach Handy beschnitten werden.
